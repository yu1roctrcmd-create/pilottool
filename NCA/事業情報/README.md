# 事業情報（AIの「脳」）

このフォルダはClaudeがNCAの業務を理解するための情報を集約しています。
Claude Codeはここを参照して適切なサポートを行います。

## フォルダ構成

| フォルダ | 内容 |
|---------|------|
| 会社概要/ | NCAの基本情報、組織図、事業内容 |
| 業務フロー/ | 日常業務の手順、ワークフロー |
| チームメンバー/ | メンバー情報、担当分野 |
| KPI目標/ | 目標値、進捗管理 |
| 重要連絡先/ | 緊急時の連絡先、外部機関 |

## 更新方法

各フォルダ内のMarkdownファイルを直接編集してください。
Claude Codeは自動的に最新情報を参照します。
