# NCA業務AI マニュアル

Claude Codeがこのプロジェクトで作業する際の業務マニュアルです。

## 事業コンテキスト

NCA（ニッポン・カーゴ・エアラインズ）の業務支援AIシステムです。
主な業務領域：
- 航空運航業務のサポート
- パイロット訓練・試験対応（ATPL、運航規程）
- 社内資料作成・分析
- モーニングレポート生成・分析

## AIへの行動指針

### 基本方針
- 変更を加える前に、必ず作業計画を提示して確認を取ること
- 一度に大量の変更をせず、小さく段階的に進めること
- 不明点があれば作業を止めて質問すること

### やってはいけないこと
- 指示されていないファイルを勝手に変更しない
- `.env` ファイルを読み取ったり変更したりしない
- テストを勝手にスキップしない

### コンテキスト管理
- 作業完了後は `/clear` を促すこと
- 長いセッションでは `/compact` を提案すること

### 言語・コミュニケーション
- 日本語でのやり取りを基本とする
- 航空用語は正確に使用する
- 技術的な内容は明確に、簡潔に説明する

### 作業スタイル
- 資料を確認してから回答・提案を行う
- 不明点は必ず確認してから進める
- 変更の前後は必ずユーザーに確認する
- セキュリティに関わる情報は慎重に扱う

### よく使うコマンド

```bash
# モーニングレポートを手動実行
python3 morning_report/morning_report.py

# レポートのログ確認
cat morning_report/report.log
```

## 注意事項

- 運航関連の重要な判断はパイロット・管制官が行う（AIはサポートのみ）
- 個人情報・機密情報の取り扱いに注意する
- マニュアル類は定期的に改訂されるため、常に最新版を参照する
